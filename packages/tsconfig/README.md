# @oriflame/tsconfig

Sharable typescript configuration.

## Installation

```sh
npm install --save-dev @oriflame/tsconfig typescript
yarn add --dev @oriflame/tsconfig typescript
```

## Usage

`tsconfig.json`:

```json
{
  "extends": "@oriflame/tsconfig"
}
```

Or

```json
{
  "extends": "@oriflame/tsconfig/tsconfig.workspace.json"
}
```
