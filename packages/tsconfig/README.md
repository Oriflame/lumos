# tsconfig-oriflame

Sharable typescript configuration.

## Installation

```sh
npm install --save-dev tsconfig-oriflame typescript
yarn add --dev tsconfig-oriflame typescript
```

## Usage

`tsconfig.json`:

```json
{
  "extends": "tsconfig-oriflame"
}
```

Or

```json
{
  "extends": "tsconfig-oriflame/tsconfig.workspace.json"
}
```
